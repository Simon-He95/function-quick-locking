此项目不在维护，因为决定合并到了fast-jump中，考虑到功能上的一致性，以及代码的复用性，和快捷键公用的问题，所以决定合并到了fast-jump中，欢迎大家使用fast-jump，如果有什么问题，可以提issue，我会尽快解决。
